# 🎓 Personalized Learning Path Generator

🚀 **AI-Based Smart Learning System for Students & Professionals**

---

## 📌 Overview

The **Personalized Learning Path Generator** is an AI-powered web application designed to create **customized learning roadmaps** based on user skills, interests, and goals.

In today’s digital era, learners are overwhelmed with excessive resources. This system solves that problem by providing a **structured, intelligent, and goal-oriented learning path**.

## ▶️ How to Run Locally

### Step 1: Clone Repository

```bash
git clone https://github.com/darpan1112/Personalized-Learning-Path-Generator.git
```

### Step 2: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 3: Run Application

```bash
python app.py
```

### Step 4: Open Browser

```
http://127.0.0.1:5000
```
